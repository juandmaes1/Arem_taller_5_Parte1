## Actividades realizadas en clase  

Durante la sesión de clase se realizó el análisis de seguridad del sistema EdukIT utilizando el modelo STRIDE.  

Se discutió en equipo cuál era el flujo más crítico del sistema, concluyendo que el proceso de autenticación (login) era el más relevante, ya que controla el acceso a información sensible como datos personales, cursos y calificaciones.  

Se identificaron los principales componentes involucrados en este flujo, como el módulo de autenticación, la gestión de sesiones, las APIs y el control de acceso. A partir de esto, se aplicó el modelo STRIDE para detectar amenazas en cada componente.  

Se tomaron decisiones de modelado como:  
- Enfocarse en un solo flujo crítico para profundizar el análisis  
- Asociar cada amenaza con un activo afectado  
- Evaluar impacto y probabilidad para determinar el nivel de riesgo  

Se utilizó principalmente Microsoft Excel para estructurar la tabla de análisis STRIDE, organizando la información de manera clara y sistemática.  

Durante la clase se logró desarrollar la mayor parte de la tabla STRIDE, incluyendo amenazas, escenarios de ataque, impactos y posibles mitigaciones.
